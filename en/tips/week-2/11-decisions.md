# Tip 11: Document Decisions

**Category:** Deep Dive

## Why Record Decisions?

In 3 months you'll ask yourself: "Why did I decide that again?"

Without documentation: Guesswork.
With documentation: Clarity.

## The Location

```
knowledge/
└── decisions/
    ├── 2025-01-tech-stack.md
    ├── 2025-01-pricing.md
    └── ...
```

## Format

```markdown
# Decision: [Title]

**Date:** [When]
**Context:** [Why was this decision needed?]
**Options:** [What were the alternatives?]
**Decision:** [What was decided]
**Rationale:** [Why this option]
```

## Personal AI Uses This

When you later ask "Which CRM should I use?", Personal AI says:
"On [Date] you decided on X because of Y. Has something changed?"

## Pro Tip

After `/pai:coach` for important decisions: Record the result in decisions/.
