# Tip 19: Customize Templates

**Category:** Power User

## The Template Folder

```
templates/
├── daily.md        # Template for daily notes
├── weekly.md       # Template for weekly review
├── project.md      # Template for new projects
└── goal.md         # Template for goals
```

## Why Customize Templates?

The default templates are generic. YOUR templates should fit YOU.

Example Daily Note:
- Default: Standup format
- For Freelancer: + Billing time per client
- For Creator: + Content ideas of the day
- For Founder: + Investor updates

## How to Customize

1. Open the template in `templates/`
2. Change the structure to your needs
3. Personal AI now uses your format

## Example: Daily Template for Freelancers

```markdown
# {{DATE}}

## Standup
- Yesterday:
- Today:
- Blockers:

## Time Tracking
| Client | Time | Task |
|--------|------|------|
|        |      |      |

## Notes
```
